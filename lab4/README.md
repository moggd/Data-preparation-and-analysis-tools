## Опис завдання

Створено програму для інтерактивного моделювання гармонічного сигналу ($y(t) = A \cdot \sin(\omega t + \varphi)$) з додаванням білого шуму. 

Реалізовано графічний інтерфейс за допомогою `matplotlib.widgets` (слайдери, кнопки, чекбокси) для керування амплітудою, частотою, фазою та параметрами шуму. Додано фільтрацію зашумленого сигналу за допомогою фільтра Баттерворта (`scipy.signal.butter`) з можливістю інтерактивного налаштування частоти зрізу.

## Вимоги до системи

* **ОС:** macOS / Linux / Windows
* **Інтерпретатор:** Python 3.9 або вище

## Інструкція із запуску та встановлення залежностей

1. Перейдіть у директорію лабораторної роботи:
```bash
cd lab4
```

2. Створіть та активуйте віртуальне середовище:
```bash
python3 -m venv venv
source venv/bin/activate
```

3. Встановіть необхідні бібліотеки:
```bash
pip install -r requirements.txt
```

4. Запустіть Jupyter Notebook:
```bash
jupyter notebook
```

5. Відкрийте файл `lab4.ipynb` та виконайте всі комірки. Для коректної роботи інтерактивних віджетів у середовищі використовується магічна команда `%matplotlib widget`.
