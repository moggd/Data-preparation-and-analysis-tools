**Виконав:** Микуляк Марко, група ФБ-46
**Курс:** Засоби підготовки та аналізу даних

## Опис завдання
* **Частина 1:** Завантаження, очищення (Data Cleaning) та аналіз даних індексу VHI для адміністративних областей України за допомогою модуля urllib та бібліотеки pandas.
* **Частина 2:** Аналіз датасету споживання електроенергії (Individual Household Electric Power Consumption). Включає складну фільтрацію даних, профілювання часу виконання процедур за допомогою timeit, стандартизацію, нормування та One Hot Encoding.

## Вимоги до системи
* **ОС:** macOS / Linux / Windows
* **Інтерпретатор:** Python 3.9 або вище

## Інструкція із запуску та встановлення залежностей

1. Перейдіть у директорію лабораторної роботи:
       cd lab2

2. Створіть та активуйте віртуальне середовище (команди для macOS):
       python3 -m venv venv
       source venv/bin/activate

3. Встановіть необхідні бібліотеки з файлу залежностей:
       pip install -r requirements.txt

4. Запустіть Jupyter Notebook:
       jupyter notebook

5. Відкрийте файли lab2_part1.ipynb та lab2_part2.ipynb. Усі скрипти для завантаження даних інтегровані безпосередньо в код ноутбуків (дані не зберігаються в репозиторії). Виконуйте комірки послідовно.
