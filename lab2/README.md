# Лабораторна робота №2: Аналіз VHI та Електроспоживання

**Виконав:** Микуляк Марко, група ФБ-46

**Курс:** Засоби підготовки та аналізу даних

## Опис завдання

* **Частина 1:** Завантаження, очищення (Data Cleaning) та аналіз даних індексу VHI для адміністративних областей України.

* **Частина 2:** Аналіз датасету споживання електроенергії (Individual Household Electric Power Consumption), складна фільтрація даних, стандартизація, нормування та One Hot Encoding.

## Вимоги до системи

* **ОС:** macOS / Linux / Windows

* **Інтерпретатор:** Python 3.9 або вище

## Інструкція із запуску та встановлення залежностей

1. Перейдіть у директорію лабораторної роботи:

        cd lab2

2. Створіть та активуйте віртуальне середовище:

        python3 -m venv venv
        source venv/bin/activate

3. Встановіть необхідні бібліотеки:

        pip install -r requirements.txt

4. Запустіть Jupyter Notebook:

        jupyter notebook

5. Відкрийте файли lab2_part1.ipynb та lab2_part2.ipynb і виконуйте комірки послідовно (всі скрипти для завантаження даних вбудовано в код).
