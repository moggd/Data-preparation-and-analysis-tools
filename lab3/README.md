## Опис завдання

Обрано датасет **Automobile (UCI)**, який задовольняє всі вимоги: Multivariate, містить Categorical, Integer, Real атрибути, має пропущені значення. 
* **Посилання на датасет:** [Automobile Data Set](https://archive.ics.uci.edu/dataset/10/automobile)

Виконано очищення даних (Data Cleaning) та побудовано 6 графіків для аналізу залежностей (lineplot, scatter, гістограма на 5 діапазонів, багатовимірні візуалізації, теплова карта з коефіцієнтами кореляції).

## Вимоги до системи

* **ОС:** macOS / Linux / Windows
* **Інтерпретатор:** Python 3.9 або вище

## Інструкція із запуску та встановлення залежностей

1. Перейдіть у директорію лабораторної роботи:
```bash
cd lab3
```

2. Створіть та активуйте віртуальне середовище:
```bash
python3 -m venv venv
source venv/bin/activate
```

3. Встановіть необхідні бібліотеки:
```bash
pip install -r requirements.txt
```

4. Запустіть Jupyter Notebook:
```bash
jupyter notebook
```

5. Відкрийте файл `lab3.ipynb` і виконуйте комірки послідовно. Скрипт автоматично перевіряє наявність даних на диску і завантажує їх лише за потреби.
